You can now run 'bitbake <target>'

Common targets are:
    yoe-simple-image
    yoe-simpleiot-image
    yoe-kiosk-image
    yoe-qt6-image
    yoe-installer-image
    yoe-debug-image
    yoe-ptest-image
    yoe-sdk-image
    meta-toolchain

You can also run generated qemu images with a command like 'runqemu nographic'.

Other commonly useful commands are:
 - 'devtool' and 'recipetool' handle common recipe tasks
 - 'bitbake-layers' handles common layer tasks
 - 'oe-pkgdata-util' handles common target package tasks
